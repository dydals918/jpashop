# Spring Boot Project
스프링 부트 프로젝트

## 프로젝트 소개
JPA를 공부하기 위해 만든 간단한 쇼핑몰

## 개발환경
- 'Java 11'
- 'Spring Boot 2.4.3'
- **IDE** : Intellli J
- **Database** : H2 Database
- **ORM** : JPA
- **Dependency** : Spring JPA, Spring web, Spring validation, thymeleaf, devtools, lombok, querydsl

## 기능
- 상품 등록, 삭제, 수정
- 상품 주문, 취소
- 회원 등록
